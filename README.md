# Elite Suraksha Phase 2

Production-oriented monorepo for the Guidewire DevTrails Phase 2 submission.

## Apps

- `apps/web` - Worker/admin web application
- `apps/api` - Backend API and business logic
- `apps/mobile` - Mobile application
- `packages/shared` - Shared schemas, types, constants

## Initial scripts

- `npm run dev:web`
- `npm run dev:api`
- `npm run dev:mobile`

## Phase 2 goals

- Real authentication
- Real backend APIs
- Real database persistence
- Weekly policy pricing
- Parametric trigger monitoring
- Claims and payout workflow
- Fraud detection
- Shared logic for web and mobile
